# 🚀 DevAssist AI – Developer Code Analysis Platform

DevAssist AI is an AI-powered developer assistant that analyzes code snippets and provides intelligent feedback such as explanations, bug detection, improvement suggestions, commit message generation, and code complexity evaluation.

---

## 🧠 Features

- 🔍 Code Explanation
- 🐞 Bug Detection
- ✨ Code Improvement Suggestions
- 📝 Git Commit Message Generation
- 📊 Code Complexity Scoring

---

## 🏗️ Architecture

Frontend (Streamlit) → Backend API (FastAPI) → AI Engine (Transformers)

- **Frontend**: Interactive UI built using Streamlit  
- **Backend**: REST API using FastAPI  
- **AI Engine**: NLP models using HuggingFace Transformers  

---

## ⚙️ Tech Stack

- Python
- FastAPI
- Streamlit
- HuggingFace Transformers
- Torch

---

## 🚀 How It Works

1. User inputs code in the UI  
2. Request is sent to FastAPI backend  
3. Backend processes using AI model  
4. Response is returned and displayed  

---

## 📌 Use Cases

- Helps beginners understand code
- Assists developers in debugging
- Improves code quality with AI suggestions
- Automates commit message generation
